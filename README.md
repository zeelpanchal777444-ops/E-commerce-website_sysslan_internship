## 💻 Built With

- CSS

---

## ✨ What's Inside

- **Home Page** — Hero section with navbar
- **Product Page** — Products shown in a grid layout
- **Cart Page** — Shows all added items
- **Login Page** — Simple login form
- **Contact Page** — Contact form

---

## ⭐ Key Features

- 🛒 **Cart Counter** — Navbar shows how many items are in the cart, updates automatically
- 🎨 **Add to Cart Button** — Button color changes when you click it
- 📱 **Responsive** — Works on mobile and desktop
- ✅ **Form Validation** — Login and contact forms check for empty fields

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. That's it — no setup needed!

```bash
cd ecommerce-website
open index.html
```
