## ⭐ Key Features

### 🛒 Live Cart Counter
Every time a product is added to cart, the navbar counter updates **instantly** — no page reload needed. This is achieved using JavaScript DOM manipulation.

### 🎨 Dynamic Add to Cart Button
When a user clicks **Add to Cart**:
- Button color changes immediately
- Button text updates to "Added ✓"
- User gets instant visual feedback

This is done using JavaScript's `classList.add()` method.

### 📱 Fully Responsive Design
The website works perfectly on:
- Mobile phones
- Tablets
- Desktop screens

### ✅ Form Validation
Both Login and Contact forms check for empty fields before submission — improving data quality and user experience.
